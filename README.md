# ECG Arrhythmia Classification Using Transfer Learning

## Overview

This project presents an automated ECG (Electrocardiogram) Arrhythmia Classification system using Deep Learning and Transfer Learning techniques.

The objective is to classify ECG images into different cardiac conditions and support intelligent healthcare diagnostics.

---

## Key Features

- ECG Image Classification
- Data Augmentation
- Dataset Balancing
- Transfer Learning
- Performance Evaluation
- ROC Curve Analysis
- Confusion Matrix Visualization

---

## Dataset

The ECG image dataset contains multiple classes representing different cardiac conditions.

### Data Preparation

- Image Size Verification
- Data Augmentation using ImgAug
- Dataset Balancing
- Train/Test/Validation Split

---

## Deep Learning Models

### MobileNetV2

Lightweight CNN architecture optimized for efficient classification.

**Accuracy:** 95.83%

---

### DenseNet201

Dense Convolutional Neural Network with feature reuse capability.

**Accuracy:** 97.22%

---

### ResNet50

Residual Neural Network architecture.

**Accuracy:** 44.33%

---
